IPython 基本

  1. Tab 自动补齐
  2. 变量？ 查看变量通用信息
  3. 函数名？ 查看函数docstring
  4. 函数名？？ 查看函数源代码
  5. %run xxx.py
  6. %paste, %cpaste
  7. Ctrl+L 清屏
  8. %debug, %pdb
  9. %hist
  10. %quickref
  11. %magic

IPython 调试

  1. %debug 进入调试器
  2. u(up), d(down) 在栈跟踪的各个级别之间切换
  3. %pdb 出现异常后自动调用调试器
  4. %run -d xxx.py, s(step)进入脚本， b(break)设置断点， c(continue)是脚本一直运行直到断点，n(next)执行下一行

IPython 性能

  1. %time, %timeit
  2. %prun, %run -p
  3. %lprun 逐行分析性能

重要的Python库 (for data analysis)

  1. NumPy

    • ndarray: N维数组对象
      • ndarray.shape
      • ndarray.dtype
      • np.array()
      • np.arange()

  2. pandas

    • Series： 类似于一维数组对象/有序字典
      • Series.values
      • Series.index
    • DataFrame: 表格行数据，它的每一列就是一个Series
      • DataFrame（dict_data/二维ndarray/Series,column=[‘c1’,‘c2’,‘c3’],index=[‘r1’,‘r2’,‘r3’])

  3. 数据加载

    • pd.read_csv()
    • pd.read_table()
    • chunk
    • df.to_csv()

  4. matplotlib
