《刻意学习》 读书笔记

  • 心理表征

有目的的练习

  1. 明确的目标：小的，可以达成的，具体的
  2. 专注：注意力集中
  3. 反馈：判断自己不足的地方
  4. 走出舒适区
  5. 试着做不同的事情

走出你的舒适区，专注的方式制定目标，为达到目标制定计划，并且想出监测你进步的方法，还要保持你学习的动机

大脑的适应能力

  1. 感受压力，打破原有平衡，重新适应变化，建立新的平衡。
  2. 足够努力地锻炼，并且保持足够长的时间，那么，身体将以各种方式改变，使得努力变得容易。
  3. 改变要不断进行下去，必须不断的加码：跑得更远一些，更快一些。
  4. 循序渐进的走出你的舒适区。

心理表征

一提到蒙娜丽莎，很多人便会在脑海中“看到”那副著名油画的形象；这个形象就是蒙娜丽莎在人们脑海中的“心理表征”。有的人更详尽，更准确，甚至还能表述背景的细节。

刻意练习，来建立心理表征。

黄金标准

刻意练习的特点是什么？ 1. 不断尝试那些刚好超出当前范围能力的事物 2. 良好定义的目标：制定计划–实现一系列微小改变—将改变累积起来 3. 有意为之 4. 包含反馈，监督自己，发现错误，及时调整

如何运用刻意练习？ 1. 确定行业内的杰出人物 （客观标准） 2. 找出杰出人物与其他人的差别，是什么使他们的训练方法与其他人的训练方法有区别 3. 如果方法管用，继续下去；如果不管用，马上停下来。调整自己的练习方法。 4. 模仿杰出人物

工作中运用刻意练习

  1. “王牌训练计划”
  2. 边干边学
  3. 及时反馈

生活中运用刻意练习

  1. 找一位好导师，自己改变时，可能需要更换
    • 导师不仅教你如何练习，而且告诉你应当注意哪些特定的方面，指出你犯了哪些错误，怎样才是卓越的表现
  2. 从事有目的的练习，不是简单的漫无目的重复
  3. 保持专注
    • 每次都要把动作做正确，一次又一次不断地积累，直到每一个细节都做到卓越，而且成为你根深蒂固的习惯
  4. 更短的练习，更高的注意力
    • 保持足够的睡眠，以便保证最大限度地集中精力

没有导师怎么办？

  1. 自己设计练习方法
    • 努力做一些你无法做到的事情
    • 完成一些在你的舒适区以外的任务
    • 一而再，再而三的练习
    • 着重关注到底怎么做好它
    • 例如：一遍又一遍的看英文电影，遮住字幕，模仿学习，关注错的地方，纠正
  2. 三个F
    • Focus
    • Feedback
    • Fix it
    • 复制大师们的绘画作品，提高自己的技能；通过研究大师的作品，努力从记忆中复制，将复制品与原始作品对比，发现差别，然后纠正
    • 复制杰出人物的成就，失败，思考，再复制，再失败，再思考，如此一而再再而三地尝试，建立有效的心理表征
  3. 跨越停滞阶段
    • 弄清楚什么导致你停滞不前：犯了什么错？着眼于妨碍你前进的特定问题，不是试试这个，试试那个，寄希望于哪种管用
    • 以新的方式继续挑战，调整你的练习方法
    • 攻克你特定的弱点，想办法稍微“逼”自己一下，再一次走出舒适区
  4. 保持动机
    • 新年决心效应
    • 意志力是不存在的，存在的是你慢慢培养的习惯
    • 减肥成功的人靠的不是意志力，是重新设计的新生活，养成的新的习惯
  5. 弱化停下脚本的理由
    • 留出固定时间练习
    • 保证充足的睡眠，养成生理时钟，（起床之后，觉得神清气爽的自然醒）
  6. 强化继续前行的倾向
    • 意识到刻意练习已变成习惯，是一种投资，而不是代价
    • 觉得自己不在状态时候，逼一下自己
    • 千万不要半途而废，即使无法前进，和自己达成协议，将自己的努力回归到之前的状态/或者跨越过停滞阶段，再放弃
  7. 精心设置目标
    • 刻意练习是一种孤独的追求
    • 分解目标，不同阶段不同的练习方法，监测每个阶段自己的进步
    • 没有理由不去追求梦想

成为杰出人物的路线图

  1. 产生兴趣
    • 鼓励，导向，再到自律
  2. 变得认真
    • 只有花了数十年培养某项技能的人，才会自然而然地喜欢上这个技能
  3. 全力投入
  4. 开拓创新
    • 不断提高自己的能力，在各种道路上摸索，以寻找正确的道路，再取得的成绩上，再尝试着做不同的事情
    • 不断超越自己
